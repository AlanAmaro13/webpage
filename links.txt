experimental transmittance spectrum with model ones using Alonso's Theorical Model. Background data as well as test sample are
downloaded. The algorithm requieres a spectrum transmittance sample and a initial guess for the thickness value with an uncertanty in nm.

absortion values, carrier density. The link to enter this notebook is: https://drive.google.com/file/d/15EbgNeVvbOfmeEGj0bpYkEV0Z7w-XsVT/view?usp=drive_link
